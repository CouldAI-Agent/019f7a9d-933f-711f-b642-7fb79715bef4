# AI Chat

Ushbu ilova Flutter orqali yaratilgan zamonaviy sun'iy intellekt chat ilovasi hisoblanadi. U mobil telefonlar, planshetlar va veb brauzerlarda chiroyli ko'rinish va muammosiz ishlashni ta'minlaydi.

## Imkoniyatlari
- Foydalanuvchi va AI o'rtasidagi suhbat ekrani.
- Chiroyli va zamonaviy xabar pufakchalari (chat bubbles).
- Material Design 3 (MD3) asosidagi zamonaviy interfeys.
- Xabar jo'natilayotganda yuklanish jarayonini (loading indicator) ko'rsatish.
- Barcha platformalarga (iOS, Android, Web) to'liq moslashgan.

## Ishga tushirish

Ilovani ishga tushirish uchun quyidagi buyruqdan foydalaning:
```bash
flutter run
```

Yoki brauzerda ko'rish uchun:
```bash
flutter run -d chrome
```

## Texnologiyalar
- Flutter & Dart
- Material 3 Design
- State Management (StatefulWidget)
